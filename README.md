# Smart Parking Management Application
</br>
Smart Parking Management Application is an intelligent parking solution that leverages computer vision, web technologies, and database management to optimize parking space utilization. It allows real-time tracking of parking availability, automated vehicle detection, and seamless management of parking slots.

---

# Features

- Automated vehicle detection and slot tracking
- Real-time parking availability updates
- Admin panel for parking management
- User-friendly interface for checking available slots
---
# Tech Stack

- Frontend: ReactJS
- Backend: Python Flask
- Database: MySQL
- Computer Vision: YOLO + OpenCV
